# Password Strength Analyzer

## What this project is
A web-based tool that evaluates password strength and provides feedback.

## Why it matters
Weak passwords are one of the main reasons accounts get hacked. This project educates users on password security.

## How the tool works
Checks passwords for length, uppercase/lowercase letters, numbers, and special characters. Classifies passwords as Weak, Medium, or Strong and provides improvement suggestions.

## How to use
1. Run the Python application.
2. Open the web browser.
3. Enter a password.
4. View the strength assessment and feedback.
